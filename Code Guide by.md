#腾讯代码规范
=============
##命名规则
-------------
1.[项目命名]
    栗子：my_project_name

2.[目录命名 有复数结构时，要采用复数命名法]
    栗子：scripts，styles，images,data_models

3.[JS文件命名 参照项目命名规则]
    栗子：account_model.js

4.[CSS,SCSS文件命名 参照项目命名规则]
    栗子：retina_sprites.scss

5.[HTML文件命名 参照项目命名规则]
    栗子：error_report.html
